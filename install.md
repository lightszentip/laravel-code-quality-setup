# Install

## Code Quality Larastan

```shell
composer require --dev larastan/larastan
composer require --dev phpat/phpat
touch phpstan.neon
````

## PHP CodeSniffer

```shell
composer  require --dev "squizlabs/php_codesniffer"
touch phpcs.xml
```
